# Fraud Detection using AutoEncoder (PyOD)

## Project Overview
This project implements a deep learning-based anomaly detection model 
using PyOD's AutoEncoder to detect fraudulent credit card transactions.

## Dataset
Kaggle Credit Card Fraud Detection dataset.(https://www.kaggle.com/datasets/whenamancodes/fraud-detection)

## Model
AutoEncoder neural network.

## Results

Achieved high ROC-AUC score in detecting fraudulent transactions.
